Home

Smart Home adalah sebuah sistem yang merupakan bagian dari teknologi Internet Of Things, yang secara sederhana adalah teknologi yang memungkinkan terjadinya komunikasi antara satu mesin dengan mesin lainnya. di indonesia sendiri teknologi ini terbilang cukup baru sehingga masih sedikit orang atau masyarakat yang tahu dan mengadopsi teknologi ini. sistem ini saya buat selain untuk memudahkan pemilik rumah dalam melakukan monitoring jarak jauh dengan kondisi rumahnya secara real-time juga sekaligus memperkenalkan teknologi Internet Of Things kepada masyarakat awam, dengan harapan kedepannya teknologi ini makin banyak diadopsi pada bidang-bidang lainnya seperti Smart City dan Smart Farming.


Cara Kerja Sistem


Cara kerja system dibagi menjadi dua, yaitu dari sisi rumah dan sisi web app.
Untuk cara kerja dari sisi rumah, pada saat dihidupkan raspberry pi akan langsung menjalankan script untuk melakukan monitoring kondisi rumah melalui data yang ditangkap sensor, mengirimkan notifikasi melalui telegram jika terdeteksi kebocoran GAS elpiji dan akan mengeksekusi perintah pada relay sesuai dengan inputan user melalui web app yang tersimpan di firebase. Sedangkan cara kerja dari sisi web app, untuk keamanan, pengguna diharuskan login terlebih dahulu, setelah login halaman akan dialihkan ke halaman control panel yang terdapat dashboard untuk monitoring kondisi rumah melalui sensor, serta terdapat delapan tombol ON/OFF untuk mengontrol switch di rumah sesuai dengan inputan yang diberikan oleh pengguna. Keseluruhan sistem baik dari sisi rumah maupun web app, semua terintegrasi dengan firebase real-time database.

Teknologi Yang Digunakan

Firebase

Firebase mempunyai peran sebagai tempat penampung data utama yang memungkinkan Raspberry Pi dan web app berkomunikasi.


Python

Bahasa Python digunakan dalam membuat sistem Smart Home agar bisa terintegrasi dengan firebase dari sisi Raspberry Pi.


Telegram

Penggunaan telegram di dalam sistem ini untuk mengirimkan notifikasi kepada pengguna jika terdeteksi kebocoran gas.


Javascript

digunakan untuk menghubungkan firebase dengan web app melalui API, berperan untuk mengirim dan menampilkan data real-time.


Raspberry Pi

Raspberry Pi adalah otak dari sistem dalam melakukan kontrol secara menyeluruh pada sistem yang ada di rumah.


Sensor

Sensor bertugas untuk melakukan monitoring kondisi rumah dan datanya akan ditampilkan pada web app secara real-time.


Switch

Switch berfungsi untuk mengatur kondisi elektronik didalam rumah dengan menerima perintah input biner 1/0 dari Raspberry Pi.


Raspbian OS

Sistem Operasi yang digunakan pada Raspberry Pi, berbasis linux dan mempunyai dukungan library yang cukup banyak.


PWA

Progressive Web App memberikan pengalaman website dan aplikasi menjadi satu tanpa harus memasang aplikasi tersebut.
